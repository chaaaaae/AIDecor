- 🎨 인터랙티브 디자인 인터페이스
- 📝 맞춤형 공간 디자인 제안

## 🚀 시작하기

### 설치
```bash
# 저장소 클론
git clone https://github.com/chaaaaae/AIDecor_ikors.git

# 프로젝트 디렉토리로 이동
cd AIDecor_ikors
```

### 실행
웹 브라우저에서 HTML 파일을 직접 열어 사용
